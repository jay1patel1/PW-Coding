Ordered List of HTML Tag and description about it

  • Document Structure
    1. <!DOCTYPE html>: Document type declaration.
    2. <html>: Root element.
    3. <head>: Contains metadata about the HTML document.
    4. <title>: Sets the title of the HTML document.
    5. <body>: Contains the content of the HTML document.
    6. <style>: Add inline css.
  • Text Formatting Tags
    1. <h1> to <h6>: Headings of different levels.
    2. <p>: Paragraph.
    3. <strong>: Represents strong importance (typically bold).
    4. <em>: Represents emphasized text (typically italic).
    5. <br>: Line break.
    6. <hr>: Horizontal rule (line).
    7. <u>: Underline.
    8. <i>: italic.
    9. <small>: for small text.
    10. <sub>: Subline text.
    11. <sup>: Superline text.
    12. <mark>: Mark/Highlight text.
    13. <del>: checked text (deleted text).
    14. <ins>: Horizontal rule (line).
    15. <ol>: Order List.
    16. <ul>: Unorder List.
    17. <table>: Table.
    18. <tr>: Table row.
    19. <td>: Table Data.
  • Linking tag
    1. <a>: Anchor (creates hyperlinks).
    2. <href>: To add hyperlinks (it is attribute by the way just added for ref).
